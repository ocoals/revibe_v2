# 개인정보처리방침

> **시행일:** 2026년 ○월 ○일
> **서비스명:** ClosetIQ (클로젯아이큐)
> **운영자:** [사업자명] (이하 "회사")

ClosetIQ(이하 "서비스")는 「개인정보 보호법」 제30조에 따라 이용자의 개인정보를 보호하고, 이와 관련한 고충을 신속하고 원활하게 처리하기 위하여 다음과 같은 개인정보 처리방침을 수립·공개합니다.

## 1. 수집하는 개인정보 항목 및 수집 방법

### 1.1 필수 수집 항목

| 수집 시점 | 항목 | 수집 목적 |
|-----------|------|----------|

| 항목 | 수집 목적 |

## 3. 이미지 데이터의 처리에 관한 특칙

서비스의 핵심 기능이 이미지 데이터를 기반으로 하므로, 이에 대해 상세히 안내합니다.

### 3.1 이미지 처리 흐름

```
이용자가 사진 업로드
  → 서버에서 EXIF 메타데이터 완전 제거 (GPS 위치정보 포함)
  → 배경 제거 처리 (의류 아이템만 분리)
  → 색상 추출 (코드 로직, AI 미사용)
  → 원본 및 처리 이미지 암호화 저장
```

### 3.2 이미지 저장 및 접근

| 이미지 유형 | 저장 위치 | 접근 권한 | 보존 기간 |
|-------------|----------|----------|----------|
| 원본 전신 사진 | Cloudflare R2 (암호화) | 본인만 접근 가능 (서명 URL, 1시간 만료) | 회원 탈퇴 시 즉시 삭제 |
| 배경 제거 아이템 | Cloudflare R2 (CDN) | 본인만 접근 가능 | 회원 탈퇴 시 즉시 삭제 |
| 룩 재현 레퍼런스 | Cloudflare R2 (암호화) | 본인만 접근 가능 | 업로드 후 6개월 경과 시 자동 삭제, 또는 회원 탈퇴 시 즉시 삭제 |
| 데일리 코디 사진 | Cloudflare R2 (암호화) | 본인만 접근 가능 | 회원 탈퇴 시 즉시 삭제 |

### 3.3 이미지의 제3자 제공 — AI 분석

- 룩 재현 기능 이용 시, 레퍼런스 이미지가 **Anthropic사의 Claude AI API**에 전송되어 패션 아이템 분석이 수행됩니다.
- 전송되는 데이터: 레퍼런스 이미지 1장 (이용자의 개인정보가 아닌 패션 코디 이미지)
- **Anthropic사는 API를 통해 전송된 데이터를 AI 모델 학습에 사용하지 않습니다.** (Anthropic API Terms of Service 기준)
- 분석 결과(의류 카테고리, 색상, 스타일 등의 텍스트 데이터)만 서버에 저장되며, 전송된 이미지는 Anthropic 측에서 분석 완료 후 보존하지 않습니다.

### 3.4 얼굴 인식 관련

- 본 서비스는 **얼굴 인식(Face Recognition) 기술을 사용하지 않습니다.**
- 배경 제거 처리는 의류 영역만을 분리하는 것으로, 이용자의 얼굴이나 생체 정보를 식별·저장·분석하지 않습니다.
- 업로드된 사진에 얼굴이 포함되어 있더라도, 배경 제거 후 의류 부분만 저장됩니다.

### 3.5 EXIF 메타데이터 제거

- 이용자가 업로드한 모든 이미지에서 EXIF 메타데이터를 **서버 수신 즉시 완전 제거**합니다.
- 제거 대상: GPS 위치 정보, 촬영 일시, 카메라 기기 정보 등

## 5. 개인정보의 파기 절차 및 방법

### 5.1 파기 절차

회원 탈퇴 요청 시 다음 절차를 따릅니다:
1. 데이터베이스에서 회원 계정 및 관련 데이터 삭제 (CASCADE 삭제)
2. 이미지 저장소(Cloudflare R2)에서 해당 회원의 모든 이미지 파일 삭제
3. 분석 도구(PostHog)에서 사용자 데이터 삭제 요청
4. 삭제 완료 확인

### 5.2 파기 방법

- 전자적 파일: 복원이 불가능한 방법으로 영구 삭제
- 이미지 파일: 저장소에서 즉시 삭제 후 CDN 캐시 무효화

### 5.3 파기 소요 시간

- 계정 및 데이터베이스: 탈퇴 요청 즉시 (실시간)
- 이미지 파일: 탈퇴 요청 후 24시간 이내
- CDN 캐시: 탈퇴 요청 후 72시간 이내 무효화
